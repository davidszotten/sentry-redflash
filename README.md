# sentry-redflash

An extension for Sentry which integrates with [Redflash](https://github.com/aquamatt/RedFlash). (SMS/twitter
notifications)

## Installation
`$ pip install sentry-redflash`

Add `sentry_redflash` to your `INSTALLED_APPS`:

```python
INSTALLED_APPS = (
    #...
    'sentry',
    'sentry_redflash',
)
```
